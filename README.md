# MEET and TASTE — Site Web
 
## 📁 Structure des fichiers
 
```
meet-and-taste/
├── index.html       ← Structure HTML complète
├── style.css        ← Tous les styles (DA premium)
├── script.js        ← Interactions, slideshow, animations
├── facade1.jpg      ← ⚠️ À ajouter : photo façade (soir/nuit)
├── facade2.jpg      ← ⚠️ À ajouter : photo façade (jour)
└── README.md
```
 
## 🖼️ Photos à placer dans le dossier
 
**IMPORTANT** : Le site utilise deux photos de la façade MEET and TASTE.  
Tu dois sauvegarder tes images générées sous ces noms exacts :
 
| Nom fichier | Image |
|-------------|-------|
| `facade1.jpg` | Photo façade version soir (fond sombre, lumières chaudes) |
| `facade2.jpg` | Photo façade version jour (fond clair, lumière naturelle) |
 
> Ces deux images sont utilisées dans :
> - Le **slideshow hero** (avec transition automatique toutes les 4s)
> - La **section concept** (côté droit)
> - La **section à propos** (photos superposées)
 
## 🚀 Déploiement GitHub Pages
 
1. Crée un repo GitHub nommé `meet-and-taste`
2. Upload tous les fichiers **y compris les photos**
3. Va dans `Settings > Pages > Source = main branch`
4. Ton site sera disponible sur `https://[ton-pseudo].github.io/meet-and-taste/`

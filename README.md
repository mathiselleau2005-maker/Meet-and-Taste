## 🚀 Mise en ligne sur GitHub Pages (étape par étape)
 
### Étape 1 — Créer le repo
1. Va sur [github.com](https://github.com)
2. Clique sur **"New repository"**
3. Nomme-le : `meet-and-taste`
4. Laisse-le **Public**
5. Clique **"Create repository"**
### Étape 2 — Uploader les fichiers
1. Dans le repo, clique **"uploading an existing file"**
2. Glisse-dépose les 3 fichiers : `index.html`, `style.css`, `script.js`
3. En bas, clique **"Commit changes"**
### Étape 3 — Activer GitHub Pages
1. Va dans **Settings** (onglet en haut du repo)
2. Dans le menu gauche, clique **Pages**
3. Sous *Source*, sélectionne **Deploy from a branch**
4. Branche : **main** · Dossier : **/ (root)**
5. Clique **Save**
### Étape 4 — Accéder au site
Après ~2 minutes, ton site est disponible à :
```
https://[ton-pseudo-github].github.io/meet-and-taste/
```
